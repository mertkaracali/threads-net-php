# threads-net-php
Threads.net Unofficial Public API with PHP




Usage

GET USER
$thread = new Threads();

$thread->get_user();

GET USER THREADS
$thread->get_user_threads();

GET USER REPLY
$thread->get_user_reply();

GET THREADS

$thread->get_threads();

GET THEADS LIKERS

$thread->get_threads_likers();
